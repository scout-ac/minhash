# MinHash

This is a fork of the original at <https://github.com/gosom/go-minhash>.

## Introduction

This is an implementation of the Minhash algorithm as descibed in chapter 3 of Mining Massive Datasets (<http://infolab.stanford.edu/~ullman/mmds/ch3.pdf>).

Implementation is inspired from the python repository <https://github.com/ekzhu/datasketch>.
